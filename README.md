# GymRotate Pro

Planificador de rotaciones de gimnasia artística. Funciona en el browser con sincronización en la nube via Firebase.

## Funcionalidades

- Drag & drop + click en celda para asignar grupos
- Modo lectura pública — cualquiera con el link ve el horario
- Modo edición — iniciá sesión para modificar
- Sincronización en tiempo real con Firebase Firestore
- Offline — los cambios se guardan localmente si no hay conexión
- Modo claro/oscuro, zoom, filtro por grupo
- Exportar PDF, PNG, CSV, iCal, JSON backup
- Undo/redo, plantillas, copiar día
- Detección de conflictos automática
- Estadísticas por grupo, profesor y aparato
